直接求b的阶乘是不可能的,递归达到最大限制次数也求不出来.

这里要用到威尔逊定理

判定一个数字是素数的充分必要条件:

(p-1)! = -1 (mod p)

在这套题中可以发现(-1)(A-1)(A-2)...(B+2)(B+1)是B! 关于A的逆元,然后就好求了


import gmpy2
import sympy
from Crypto.Util import number

def myprime(a, b):
    ans = 1
    for i in range(b+1, a):
        ans = (ans *i) % a
    ans = -1 * ans
    num = gmpy2.invert(ans ,a)
    return sympy.nextprime(num)
